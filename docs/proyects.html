Proyectos

1 Proyecto

Este proyecto es una aplicación web moderna creada utilizando Sass, HTML y Vite. El objetivo es aprovechar la velocidad de Vite para un desarrollo ágil, combinando Sass para gestionar estilos de manera avanzada y estructurada. La interfaz tiene un diseño limpio y atractivo, con transiciones y animaciones suaves que mejoran la experiencia del usuario. Utilizando Sass, los estilos están organizados con variables y mixins, facilitando su mantenimiento y escalabilidad. Todo el código está optimizado para una carga rápida en todos los dispositivos. Además, el proyecto está disponible en GitHub, donde se puede acceder al código fuente y seguir su evolución.

2 Proyecto

Este proyecto es una aplicación web dinámica desarrollada con Java, JSP, JDBC y una base de datos SQL. El objetivo principal es gestionar y presentar información en tiempo real, conectando la interfaz de usuario con una base de datos relacional. Mediante JDBC, se implementa una comunicación eficiente y segura entre el servidor y la base de datos, permitiendo realizar operaciones CRUD (Crear, Leer, Actualizar y Eliminar). La interfaz, creada con JSP, combina la lógica de negocio con un diseño funcional, proporcionando una experiencia interactiva. El proyecto destaca por su arquitectura bien estructurada y su código optimizado, disponible en GitHub para explorar cada detalle.
